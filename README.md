## Sobre

Plataforma educacional completa do **Liceu Monteiro Lobato**. Quatro portais integrados: alunos, professores, pais e administracao. Centraliza notas, frequencia, comunicados e agenda escolar em um unico ambiente digital.

## Portais

| Portal | Publico | Funcionalidades |
|--------|---------|-----------------|
| Alunos | Estudantes | Notas, horarios, materiais |
| Professores | Docentes | Lancamento de notas, frequencia |
| Pais | Responsaveis | Acompanhamento e comunicados |
| Admin | Gestao | CRUD completo, relatorios |

## Stack

| Camada | Tecnologia |
|--------|------------|
| Framework | Next.js 14, React 18 |
| Linguagem | TypeScript strict |
| Estilos | TailwindCSS 3.4 |

## Comecando

```bash
git clone https://github.com/marcelogourod/liceu-monteiro-lobato.git
cd liceu-monteiro-lobato
pnpm install
pnpm dev
```
